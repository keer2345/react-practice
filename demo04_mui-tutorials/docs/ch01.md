# Installation

```sh
yarn add @mui/material @emotion/react @emotion/styled
yarn add @mui/icons-material
```

`index.css`:

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
  font-family: Inter, system-ui, Avenir, Helvetica, Arial, sans-serif;
}
```
